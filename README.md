# 🤖 AI Image Detector

## 📌 Overview

AI Image Detector is a machine learning project that detects whether an uploaded image is AI-generated or human-created.

## 🚀 Features

- Detect AI-generated images
- Detect real (human-created) images
- Metadata analysis
- Deep Learning prediction
- User-friendly interface

## 🛠 Technologies Used

- Python
- Google Colab
- TensorFlow
- NumPy
- Pillow
- OpenCV
- Matplotlib

## 📂 Project Structure

```
AI-Image-Detector
│
├── AI_IMAGE_PROGRAM.ipynb
├── README.md
└── requirements.txt
```

## 📖 How to Run

1. Clone the repository.
2. Install the required libraries.
3. Open the notebook in Google Colab or Jupyter Notebook.
4. Run all cells.
